# Kurzanleitung - Kostenlose Version

Diese Anleitung ist eigenstaendig und benoetigt keine externe Website.

## Voraussetzungen

- WordPress 7.0+
- PHP 8.0+
- Mindestens ein konfigurierter und aktiver WordPress-AI-Connector
- Benutzer mit Berechtigung zum Bearbeiten von Beitraegen

Wichtig: Ohne aktiven AI-Connector kann das Plugin aktiviert werden, aber keine Entwuerfe aus Audio generieren.

## 1) Installation

1. Plugin-Ordner nach wp-content/plugins kopieren.
2. Audio Converter for WP aktivieren.
3. Zu Einstellungen > Audio Converter gehen.
4. Standardwerte speichern.

## 2) Empfohlene Konfiguration

- Standardsprache: Hauptsprache der Website
- Standardton: professional
- Standardlaenge: medium
- Hinweise zu Eigennamen: Marken, Personen, Orte, Produkte
- Einfuegemodus: append (empfohlen)

## 3) Nutzung im Block-Editor

1. Beitrag oeffnen oder neu erstellen.
2. Audio Converter Sidebar oeffnen.
3. Auf Select audio from Media Library klicken.
4. Eine Audiodatei waehlen.
5. Redaktionelle Optionen pruefen.
6. Auf Generate draft from audio klicken.

## 4) Erwartetes Ergebnis

- Das Plugin fuegt Gutenberg-Bloecke in den aktuellen Beitrag ein.
- Wenn aktiviert, wird der generierte Titel uebernommen.
- Bei Fehlern erscheint eine Meldung in der Sidebar.

## 5) Schnelle Fehlerbehebung

- Sidebar nicht sichtbar: Plugin aktiv und Block-Editor aktiv?
- AI-Fehler: AI-Connectoren pruefen.
- Keine Bloecke eingefuegt: Fehlermeldung pruefen und klareres Audio verwenden.

## 6) Best Practices

- Klares Audio mit wenig Hintergrundgeraeuschen verwenden.
- Hinweise zu Eigennamen eintragen.
- Fuer bessere Ergebnisse kurze, deutliche Aufnahmen nutzen.
