# Quick Start Guide - Free Version

This guide is self-contained and does not require an external website.

## Requirements

- WordPress 7.0+
- PHP 8.0+
- At least one WordPress AI connector configured and active
- User role with post editing capability

Important: without an active AI connector, the plugin can be activated but cannot generate drafts from audio.

## 1) Installation

1. Copy the plugin folder to wp-content/plugins.
2. Activate Audio Converter for WP.
3. Go to Settings > Audio Converter.
4. Save your defaults.

## 2) Recommended defaults

- Default language: your main site language
- Default tone: professional
- Default target length: medium
- Proper noun hints: brands, people, places, products
- Editor insertion mode: append (recommended)

## 3) Use in the Block Editor

1. Open or create a post.
2. Open the Audio Converter sidebar.
3. Click Select audio from Media Library.
4. Select one audio file.
5. Review editorial options.
6. Click Generate draft from audio.

## 4) What to expect

- The plugin inserts Gutenberg blocks in the current post.
- If enabled, generated title is applied to the post title.
- Errors are shown directly in the sidebar.

## 5) Quick troubleshooting

- Sidebar not visible: confirm plugin is active and you are in Block Editor.
- AI provider error: verify AI connector configuration.
- No blocks inserted: check sidebar message and retry with cleaner audio.

## 6) Best practices

- Use clear audio with limited background noise.
- Add proper noun hints for names and brands.
- Prefer short and clear recordings for better output quality.
