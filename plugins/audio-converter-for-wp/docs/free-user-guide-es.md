# Guia rapida - Version gratuita

Esta guia es autonoma y no requiere un sitio web externo.

## Requisitos

- WordPress 7.0+
- PHP 8.0+
- Conectores de IA de WordPress configurados
- Usuario con permisos para editar entradas

## 1) Instalacion

1. Copia la carpeta del plugin en wp-content/plugins.
2. Activa Audio Converter for WP.
3. Ve a Ajustes > Audio Converter.
4. Guarda los valores por defecto.

## 2) Configuracion recomendada

- Idioma por defecto: idioma principal del sitio
- Tono por defecto: professional
- Longitud objetivo por defecto: medium
- Sugerencias de nombres propios: marcas, personas, lugares, productos
- Modo de insercion: append (recomendado)

## 3) Uso en el editor de bloques

1. Abre o crea una entrada.
2. Abre la barra lateral de Audio Converter.
3. Haz clic en Select audio from Media Library.
4. Selecciona un archivo de audio.
5. Revisa las opciones editoriales.
6. Haz clic en Generate draft from audio.

## 4) Resultado esperado

- El plugin inserta bloques Gutenberg en la entrada actual.
- Si esta activo, el titulo generado se aplica al titulo de la entrada.
- En caso de error, se muestra un mensaje en la barra lateral.

## 5) Solucion rapida de problemas

- Barra lateral no visible: verifica que el plugin este activo y que uses el editor de bloques.
- Error del proveedor de IA: verifica la configuracion de los conectores de IA.
- No se insertan bloques: revisa el mensaje de error e intenta con audio mas claro.

## 6) Buenas practicas

- Usa audio claro y con poco ruido de fondo.
- Anade sugerencias de nombres propios para mejorar precision.
- Para mejores resultados, usa clips cortos y bien articulados.
