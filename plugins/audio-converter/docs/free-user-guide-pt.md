# Guia rapida - Versao gratuita

Este guia e independente e nao precisa de site externo.

## Requisitos

- WordPress 7.0+
- PHP 8.0+
- Pelo menos um conector de IA do WordPress configurado e ativo
- Usuario com permissao para editar posts

Importante: sem um conector de IA ativo, o plugin pode ser ativado, mas nao consegue gerar rascunhos a partir de audio.

## 1) Instalacao

1. Copie a pasta do plugin para wp-content/plugins.
2. Ative o Audio Converter.
3. Acesse Configuracoes > Audio Converter.
4. Salve os valores padrao.

## 2) Configuracao recomendada

- Idioma padrao: idioma principal do site
- Tom padrao: professional
- Tamanho alvo padrao: medium
- Dicas de nomes proprios: marcas, pessoas, lugares, produtos
- Modo de insercao: append (recomendado)

## 3) Uso no editor de blocos

1. Abra ou crie um post.
2. Abra a barra lateral do Audio Converter.
3. Clique em Select audio from Media Library.
4. Selecione um arquivo de audio.
5. Revise as opcoes editoriais.
6. Clique em Generate draft from audio.

## 4) Resultado esperado

- O plugin insere blocos Gutenberg no post atual.
- Se ativado, o titulo gerado e aplicado ao titulo do post.
- Em caso de erro, aparece uma mensagem na barra lateral.

## 5) Solucao rapida de problemas

- Barra lateral nao aparece: verifique se o plugin esta ativo e se voce esta no editor de blocos.
- Erro do provedor de IA: verifique os conectores de IA.
- Nenhum bloco inserido: confira a mensagem de erro e tente com audio mais claro.

## 6) Boas praticas

- Use audio claro e com pouco ruido de fundo.
- Adicione dicas de nomes proprios para maior precisao.
- Para melhores resultados, use gravacoes curtas e bem articuladas.
