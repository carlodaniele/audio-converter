# FAQ - Version gratuite

## Faut-il un site externe avec des tutoriels?

Non. La documentation essentielle est incluse dans le plugin.

## Puis-je l'utiliser sans API externe personnalisee?

Oui. Utilisez les connecteurs IA WordPress deja configures.

## Cree-t-il un nouvel article?

Dans le flux editeur, il met a jour l'article en cours.

## Quel type d'audio est recommande?

Un audio clair, avec volume stable et peu de bruit de fond.

## Que faire si la generation echoue?

1. Verifiez les connecteurs IA.
2. Reessayez avec un audio plus court ou plus propre.
3. Verifiez le message dans la barre laterale.

## Comment ameliorer la qualite du resultat?

- Configurez correctement langue, ton et longueur.
- Ajoutez des indications de noms propres.
- Evitez le bruit et les voix qui se chevauchent.
