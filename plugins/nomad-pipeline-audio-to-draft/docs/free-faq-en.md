# FAQ - Free Version

## Do I need an external tutorial website?

No. Core documentation is included in the plugin package.

## Can I use it without custom external APIs?

Yes. Use WordPress AI connectors configured on your site.

## Does it create a new post?

In the editor flow, it updates the current post.

## Which audio files are recommended?

Use clear audio with stable volume and low background noise.

## What should I do if generation fails?

1. Check AI connector configuration.
2. Retry with shorter or cleaner audio.
3. Check the error message in the sidebar.

## How can I improve output quality?

- Set language, tone, and target length correctly.
- Add proper noun hints for names, brands, and places.
- Avoid noisy recordings and overlapping speakers.
